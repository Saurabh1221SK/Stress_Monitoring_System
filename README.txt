SETUP GUIDE

1. Put your model_emosi.h5 file inside:
   backend/model_emosi.h5

2. Open terminal in backend folder

3. Install requirements:
   pip install -r requirements.txt

4. Run backend:
   python app.py

5. Open Chrome:
   chrome://extensions/

6. Enable Developer Mode

7. Click "Load Unpacked"

8. Select the extension folder

9. Open the extension popup

10. Allow camera access
